# Lucky Graves
Headstone sampling, OCR, storage, and probabilistic search.<br />
(Demo: http://olsen.org/alan/luckyGraves)

Requirements:
- Python
- OpenCV; OpenCV-Python
- Tesseract; python-tesseract
- MySql Server
- PHP
- UIKit (extracted to www)
- jQuery (placed in www/js)
